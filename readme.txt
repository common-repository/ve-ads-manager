== Description ==

"VE Ads Manager" A plugin that will help to manage your Ads on your site.

= Features =


* An options for show Ads below/above of content area.
* shortcodes for add to Ads in website sidebar, header, footer.
 
== Installation ==

Installing the plugin:

1. Upload the plugin files to the `/wp-content/plugins/ve-ads-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings ->VE Ads Manager screen to configure the plugin
